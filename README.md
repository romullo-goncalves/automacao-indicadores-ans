# 📊 Automação de Indicadores da ANS

Este projeto tem como objetivo automatizar a coleta, tratamento, monitoramento e visualização de indicadores da ANS (Agência Nacional de Saúde Suplementar), por meio da integração entre Python, Power Automate e Power BI.

## 🧩 Contexto

A ANS define e monitora indicadores essenciais de desempenho das operadoras e instituições de saúde suplementar. Este projeto automatiza a coleta desses dados, assegura a atualização recorrente e disponibiliza visualizações analíticas, apoiando a tomada de decisão com base em dados confiáveis e atualizados.

---

## 🎯 Objetivos

- Reduzir o trabalho manual de coleta e planilhamento de indicadores ANS.
- Monitorar indicadores com alertas automatizados em caso de desvios.
- Disponibilizar painéis atualizados e interativos no Power BI.
- Promover rastreabilidade e governança dos dados monitorados.

---

## ⚙️ Tecnologias Utilizadas

- **Python** – para coleta e tratamento dos dados (via APIs, planilhas ou bases internas).
- **Power Automate** – para agendamento, integração e envio de alertas.
- **Power BI** – para visualização de indicadores e desempenho.
- **SharePoint / OneDrive** – como repositório intermediário de arquivos (se aplicável).
